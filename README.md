# Automated Hospital Log Management System

## Description
This project simulates real-time hospital device monitoring and provides automated log archiving and analysis using shell scripting and Linux tools.

## Features
- Real-time device simulation
- Interactive log archiving
- Intelligent log analysis
- Timestamped archives
- CLI-based reporting

## How to Run
1. Start simulators:
   python3 heart_rate_monitor.py start
2. Archive logs:
   ./archive_logs.sh
3. Analyze logs:
   ./analyze_logs.sh

## Tools Used
- Bash
- awk, sort, uniq
- Python
- Git
